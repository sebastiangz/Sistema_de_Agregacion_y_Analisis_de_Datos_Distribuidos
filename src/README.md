# Proyecto 7  
# Sistema de Agregación y Análisis de Datos Distribuidos  

# Descripción del Proyecto  
Este proyecto implementa una versión básica del modelo MapReduce, utilizado para el procesamiento y análisis de grandes volúmenes de datos.  

Se desarrolla bajo el enfoque de programación funcional, aplicando conceptos como funciones puras, inmutabilidad y composición.  

La versión actual realiza una tarea simple: dividir texto en palabras y contar cuántas veces aparece cada una, como primer paso hacia un sistema de análisis distribuido más completo.  

# Objetivos  
- Comprender la lógica del modelo MapReduce.  
- Aplicar principios de la programación funcional.  
- Realizar operaciones de agregación simples y composables.  
- Preparar la base para procesamiento paralelo de datos.
